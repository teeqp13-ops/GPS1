# GPSPlus MM

مشروع Theos نظيف بصيغة Objective-C++ (`.mm`) مبني اعتمادًا على البنية التي ظهرت في ملفات GPSPlus المرفوعة.

## الموجود
- خريطة MapKit وبحث عن المواقع.
- تحديد الإحداثيات بالضغط المطوّل.
- حفظ الإعدادات محليًا.
- محرك محاكاة داخلي مع انحراف بسيط Drift.
- مخزن مواقع مفضلة جاهز للربط بالواجهة.
- زر عائم داخل SpringBoard لفتح الواجهة.

## مهم
المشروع لا يتضمن تجاوز ترخيص، مفاتيح خادم، إخفاء تلاعب، أو حقن موقع مزيف في تطبيقات خارجية. محرك الموقع الحالي يرسل تحديثات داخلية فقط عبر Notification، وهو مناسب كأساس لتطبيقات الاختبار المملوكة لك.

## البناء
```bash
export THEOS=~/theos
make package
```

للتثبيت على جهاز اختبار مرتبط:
```bash
make package install
```

## التوافق
- iOS 15 أو أحدث.
- arm64 وarm64e.
- Theos وMobileSubstrate/ElleKit حسب بيئة الجهاز.
