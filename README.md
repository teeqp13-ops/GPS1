# GPSPlus License System

نظام إدارة تراخيص GPSPlus باستخدام PHP وSQLite، مع لوحة تحكم عربية وواجهات API للتفعيل والفحص والنبض الدوري.

## المكونات

- لوحة إدارة الأكواد
- API: activate / check / heartbeat
- قاعدة SQLite
- عميل Objective-C++ (`GPSAllRequests.mm`)

> قبل الاستخدام عدّل `config.php` وملف العميل بالقيم الخاصة بخادمك، ولا ترفع المفاتيح السرية إلى مستودع عام.
